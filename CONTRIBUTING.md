# Contributing to @heitorpolidoro Projects

Thank you for your interest in contributing! This repository contains the default configurations for my projects.

## How to Contribute

1. **Open an Issue**: For any major changes, please open an issue first to discuss what you would like to change.
2. **Fork the Repo**: Create your own fork and work on a new branch.
3. **Submit a Pull Request**: Ensure your PR follows the PR template and includes relevant tests.

## Coding Standards
- Maintain consistency with existing configurations.
- Ensure all CI/CD workflows are tested.
- Keep documentation up to date.
