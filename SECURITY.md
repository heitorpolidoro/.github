# Security Policy

## Supported Versions

Only the latest version of these configurations is supported.

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue. Instead, report it through one of the following methods:

- **GitHub Security Advisory**: Use the "Report a vulnerability" button in the Security tab.
- **Direct Contact**: Reach out to @heitorpolidoro directly if necessary.

I will attempt to acknowledge your report within 48 hours and provide a fix as soon as possible.
